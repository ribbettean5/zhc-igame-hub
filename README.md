# zhc-igame-hub

本仓库主要用于归档和发布多个独立 HTML 页面。这些页面内容相互独立，不针对任何特定网站或域名，旨在提供一个集中存放与展示 HTML 资源的地方。

## 项目简介

zhc-igame-hub 是一个轻量的 HTML 页面归档仓库。这里存放的页面可能包含工具、小游戏、演示页面、信息展示页等，所有页面均为独立的 HTML 文件，不依赖后端服务。仓库结构简单，便于浏览和直接访问。

## 目录结构

```
zhc-igame-hub/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

目录结构可能会随内容更新而调整，但始终以保持清晰和易用为原则。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面之间相互独立，不存在跨页面依赖。
- 部分页面可能引用 `assets/` 目录下的公共资源文件。
- 归档内容会不定期更新，新增或移除页面均属正常维护。

## 使用方式

1. 直接访问本仓库的 GitHub Pages 服务（如已启用）即可浏览页面列表。
2. 或克隆仓库到本地，用浏览器打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过原始文件链接直接访问单个页面。
